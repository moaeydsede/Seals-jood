# Jood kids seals — نظام المبيعات (نسخة الفاتورة + Excel) — بدون مجلدات
Version: 2026.02.19-DIAG-FIX

## أهم الميزات المطلوبة
- فورمة فاتورة مبيعات / مرتجع مبيعات:
  - إدخال رقم الصنف -> يملأ الاسم + السعر + الوحدة تلقائياً
  - أنت تدخل الكمية فقط
  - السعر قابل للتعديل للأدمن فقط
  - خصم: نسبة/قيمة متزامنين (عند كتابة نسبة يظهر رقم والعكس)
  - عرض قبل/بعد الخصم
- استيراد/تصدير المواد Excel (SheetJS)

## Excel Import Format
الأعمدة في أول صف:
- code
- name
- price

## ملاحظة Firestore Index
هذه النسخة لا تعتمد على orderBy مع where في القوائم لتفادي متطلبات الـ Index.
